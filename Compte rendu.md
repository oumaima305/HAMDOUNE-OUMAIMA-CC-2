## 1. Introduction
Les choix d’investissement des individus dépendent d’un ensemble complexe de caractéristiques personnelles, de motivations psychologiques et d’attentes financières. Comprendre ces mécanismes est essentiel pour mieux segmenter les investisseurs et proposer des produits financiers adaptés.
Dans ce travail, nous cherchons à répondre à la problématique suivante :
Comment les caractéristiques individuelles (démographie, facteurs influençant les choix, attentes et raisons d’investissement) se combinent-elles pour déterminer les préférences d’investissement et les objectifs d’épargne ?

Les objectifs principaux de cette analyse sont :

Identifier des profils types d’investisseurs dans la base de données, selon l’âge, le sexe, les attentes et les raisons qui motivent leurs décisions.

Développer un modèle prédictif capable d’estimer l’option d’investissement privilégiée (actions, obligations, dépôts à terme, fonds communs) à partir des caractéristiques individuelles.

Cette étude permettra de mieux comprendre les drivers essentiels qui expliquent les préférences d’investissement.
